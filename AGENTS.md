# Agentes do Sistema

1. Architect Agent
- Define estrutura
- Cria boundaries
- Nunca escreve código final

2. Backend Agent
- Cria APIs
- Modela banco
- Escreve testes

3. Frontend Agent
- Cria UI funcional
- Não cria design complexo

4. Infra Agent
- Docker
- CI/CD
- Deploy

5. QA Agent
- Executa testes
- Tenta quebrar o sistema
- Bloqueia deploy se falhar

Nenhum agente pode atuar fora de seu escopo.
