# Escopo do Produto

A plataforma DEVE:

- Gerar backend completo (APIs, regras, banco)
- Gerar frontend funcional
- Criar testes automatizados mínimos
- Publicar automaticamente em produção
- Fazer rollback automático em caso de falha
- Assumir projetos existentes via repositório Git
- Diagnosticar qualidade e risco de projetos herdados
- Evoluir produtos existentes com segurança

Casos suportados:
- SaaS B2B
- Ferramentas internas
- CRUDs complexos
- Dashboards
- APIs REST

O MVP deve suportar pelo menos:
- 1 produto por vez
- 1 ambiente de produção
- Deploy automático completo
